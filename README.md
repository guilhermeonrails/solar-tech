# ☀️ SolarTech Brasil - Soluções em Energia Fotovoltaica

A **SolarTech Brasil** é uma aplicação de e-commerce moderna e simplificada para a venda de equipamentos de energia solar. Desenvolvida para o mercado brasileiro, a plataforma oferece uma experiência de utilizador fluida — desde a navegação no catálogo até à simulação de checkout com métodos de pagamento locais.

---

## 🛠️ Tecnologias Utilizadas

Para garantir leveza, desempenho e compatibilidade, o projeto foi construído com:

- **HTML5** — Estrutura semântica e acessível  
- **Tailwind CSS (via CDN)** — Design responsivo e moderno sem necessidade de ficheiros CSS pesados  
- **JavaScript (Vanilla)** — Lógica de negócio pura para gestão de estado e navegação SPA  
- **Lucide Icons** — Biblioteca de ícones minimalistas  
- **Google Fonts** — Tipografia *Plus Jakarta Sans* para uma estética premium  

---

## ✨ Funcionalidades Principais

### 🔄 Vistas Dinâmicas (SPA)
- Navegação entre:
  - Página Inicial  
  - Loja  
  - Carrinho  
  - Checkout  
- Sem recarregamento do navegador  

### 🛍️ Catálogo Inteligente
- Filtros por categoria:
  - Painéis  
  - Inversores  
  - Baterias  
  - Kits  
- Navegação rápida e intuitiva  

### 🛒 Gestão de Carrinho
- Adição e remoção de produtos  
- Atualização dinâmica de quantidades  
- Cálculo automático de totais  
- Formatação em moeda brasileira (R$)  

### 💳 Checkout Localizado
- Formulário adaptado ao Brasil:
  - CEP  
  - Cidade / UF  
- Métodos de pagamento:
  - PIX (com desconto)  
  - Boleto Bancário  
  - Cartão de Crédito  

### 📱 Responsividade Total
- Interface otimizada para:
  - Smartphones  
  - Tablets  
  - Desktops  

### 🔔 Feedback de Utilizador
- Animações no ícone do carrinho  
- Ecrã de confirmação de compra

## 📁 Estrutura de Ficheiros

```

/projeto
│── index.html   # Estrutura base e contentores dinâmicos
│── style.css    # Animações e estilos personalizados
│── script.js    # Lógica da aplicação e renderização

````

- **index.html** → Estrutura principal da aplicação  
- **style.css** → Animações (bounce, fadeIn) e tipografia  
- **script.js** → Dados, carrinho e navegação SPA
